About this tool

This fan-made tool helps you run Pokémon auction drafts for Showdown / Wi-Fi leagues. Set budgets, nominate Pokémon, bid in real time, and export results.

How it works

  • Host creates a game and shares the code.
  • Players join the lobby, pick a name and icon.
  • Each player nominates and bids on their turn.
  • Teams and budgets update live.
